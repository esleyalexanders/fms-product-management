## 8. Component Interaction Flows

### 8.1 Add Service Flow

```
User clicks "Add from Pricebook"
        ↓
Search modal opens
        ↓
User searches "house"
        ↓
Results show services & products
        ↓
User clicks "🏠 House Cleaning"
        ↓
Service added to line items
        ↓
Real-time calculation updates
        ↓
Summary sidebar updates
        ↓
Toast: "House Cleaning added"
```

### 8.2 Add Product Flow

```
User clicks "+ Add Custom Item"
        ↓
Custom item modal opens
        ↓
User selects "● Product"
        ↓
Product fields show (SKU, Stock)
        ↓
User fills: Name, Qty, Price
        ↓
User clicks "Add to Quote"
        ↓
Validation runs
        ↓
If valid → Product added to line items
        ↓
If invalid → Show errors inline
```

### 8.3 Apply Discount with Approval

```
User applies 25% discount
        ↓
System checks threshold (>20%)
        ↓
Approval required banner shows
        ↓
User enters justification
        ↓
User clicks "Submit for Approval"
        ↓
Quote saved as "Pending Approval"
        ↓
Email sent to manager
        ↓
Manager reviews & approves
        ↓
Staff notified
        ↓
Quote status → "Approved"
        ↓
"Send Quote" button enabled
```
